Билет 10 Функции

Практика

Задания
  1. Написать функцию суммы двух чисел
  2. Написать функцию проверки чётности
Примерные решения
// 1. Функция суммы двух чисел
function sum(a, b) {
  return a + b;
}

      
// 2. Функция проверки чётности
function isEven(number) {
  return number % 2 === 0;
}

Разбор кода

function test() {
  console.log("A");
}
console.log(test());
Что произойдёт:
1. Вызывается функция test()
2. Внутри выполняется console.log("A") → в консоль выводится: A
3. Функция не имеет return → автоматически возвращает undefined
4. Внешний console.log(test()) выводит то, что вернула функция → undefined

Итоговый вывод в консоль: 
A
undefined
